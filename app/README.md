# App
Frontend part of Loopr project

## Requirements
- node 12.18.3
- yarn 1.x

## Setup
1. `yarn install`

## Development
-  `yarn dev`

## Testing
- `yarn test`

## Storybook
- `yarn storybook`

## Config
- You can edit config by environment variables in cli or in `.env`

## Other
- Do not update `core-js` to version 3 - next-i18next requires version 2, it would break storybook
