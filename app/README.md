# App
Frontend part of Loopr project

## Requirements
- node 12.18.3
- yarn 1.x

## Setup
1. `yarn install`

## Development
-  `yarn dev`

## Testing
- `yarn test`

## Storybook
- `yarn storybook`

## Config
- You can edit config by setting environment variables in cli or in `.env.local` (see `.env` for config options)
